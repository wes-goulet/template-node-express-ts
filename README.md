# template-node-express-ts
Starter repo for node+express+typescript+vscode debug configs

## How to run

```sh
npm start
```

## How to debug

```sh
npm run debug
```

Now you can press F5 in VS Code to attach to nodemon, and step through code, hit breakpoints, etc.
